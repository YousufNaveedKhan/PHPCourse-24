## Task Overview

You are required to develop four PHP programs as outlined below. Ensure that each program meets the specified requirements and is properly documented. You will also need to submit your work on GitHub.

### 1. Leap Year Program

- **Objective**: Write a PHP script to determine if a given year is a leap year.
- **Requirements**:
  - Store a year in a variable.
  - Use conditional statements and operators to check if the year is a leap year.
  - If the year is a leap year, use `echo` to print "Leap Year" along with the year.
  - If the year is not a leap year, use `echo` to print "It's not a Leap Year" along with the year.
  - When printing "Leap Year", use `echo`. When printing "It's not a Leap Year", use `EcHo`.

### 2. Even or Odd Program

- **Objective**: Create a PHP script to determine if a given number is even or odd.
- **Requirements**:
  - Store a number in a variable.
  - Use conditional statements and operators to check if the number is even or odd.
  - If the number is even, print "Even".
  - If the number is odd, print "Odd".

### 3. Positive or Negative Number Program

- **Objective**: Develop a PHP script to determine if a number is positive, negative, or zero.
- **Requirements**:
  - Store a number in a variable.
  - Use conditional statements and operators to check if the number is positive, negative, or zero.
  - Print "The number is positive" if the number is greater than zero.
  - Print "The number is negative" if the number is less than zero.
  - Print "The number is neither positive nor negative" if the number is zero.

### 4. Marksheet Program

- **Objective**: Create a PHP script to display a marksheet with 8 subjects.
- **Requirements**:
  - Hard-code marks for 8 subjects.
  - Calculate the total and percentage.
  - Use variables, print statements, operators, and conditional statements.
  - Create an HTML table to display the marks, total, percentage, and status (Pass/Fail).
  - Ensure the UI is user-friendly and well-formatted.
